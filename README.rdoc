== Installation

Install the application using gem.

    gem install aethyr

== Running the app

Just run the application with the following command.

    aethyr

== From source

     bundle install
     bundle exec rake
     bundle exec ./bin/aethyr_setup
     bundle exec ./bin/aethyr run
